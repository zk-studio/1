# 精美网站

一个简洁美观的单页面网站模板，使用纯 HTML 和 CSS 构建。

## 项目简介

这是一个响应式的单页面网站，包含以下特性：
- 清晰的导航栏
- 居中的内容展示区域
- 固定在底部的页脚
- 简洁的绿色配色方案

## 技术栈

- HTML5
- CSS3

## 功能特点

- ✨ 简洁美观的界面设计
- 📱 响应式布局，适配不同屏幕尺寸
- 🎨 使用绿色主题色（#4CAF50）
- 🔗 导航菜单（首页、关于、联系）
- 💫 悬停效果

## 文件结构

```
.
└── index.html        # 主页面文件
```

## 快速开始

1. 克隆仓库
```bash
git clone https://github.com/zk-studio/1.git
```

2. 在浏览器中打开 `index.html` 文件即可查看效果

## 使用方法

直接用浏览器打开 `index.html` 文件，或者使用本地服务器：

```bash
# 使用 Python 启动简单的 HTTP 服务器
python -m http.server 8000

# 然后在浏览器中访问
# http://localhost:8000
```

## 自定义

你可以轻松自定义网站的内容和样式：

1. **修改文本内容**：编辑 HTML 标签内的文字
2. **更改颜色主题**：修改 CSS 中的 `#4CAF50`（绿色）为你喜欢的颜色
3. **调整布局**：修改 `<style>` 标签内的 CSS 样式

## 页面预览

网站包含以下部分：
- **Header（头部）**：显示网站标题
- **Navigation（导航栏）**：包含首页、关于和联系链接
- **Content（内容区）**：主要内容展示区域
- **Footer（页脚）**：版权信息
